NUnitTest Properties

The properties of the NUnitTest class are listed below. For a complete list of NUnitTest class members, see the NUnitTest Members topic.

Public Instance Properties

AppConfigFile The application configuration file to use for the NUnit test domain.
Assembly Assembly to load the test from.
Class Class name of the test.
Fork Run the tests in a separate AppDomain.
HaltOnError Stop the build process if an error occurs during the test run.
HaltOnFailure Stop the build process if a test fails (errors are considered failures as well).
Name (inherited from Element) Gets the name of the XML element used to initialize this element.
NamespaceManager (inherited from Element) Gets or sets the XmlNamespaceManager.
OutFile Base name of the test result. The full filename is determined by this attribute and the extension of formatter.
Parent (inherited from Element) Gets or sets the parent of the element.
Project (inherited from Element) Gets or sets the Project to which this element belongs.
Properties (inherited from Element) Gets the properties local to this Element and the Project.
ToDir Directory to write the reports to.
