Parameters

s
input string

Return Value

The string s in uppercase.

Remarks

The casing rules of the invariant culture are used to convert the s to uppercase.

Example

string::to-upper('testing string') ==> 'TESTING STRING'
string::to-upper('Testing String') ==> 'TESTING STRING'
string::to-upper('Test 123') ==> 'TEST 123'
