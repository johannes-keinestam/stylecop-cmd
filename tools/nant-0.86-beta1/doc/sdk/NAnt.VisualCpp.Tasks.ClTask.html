Remarks

This task is intended for version 13.00.9466 of cl.exe.

Example

Compiles helloworld.cpp for the Common Language Runtime.

    
<cl outputdir="build" options="/clr">
    <sources>
        <include name="helloworld.cpp" />
    </sources>
</cl>
